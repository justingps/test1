About This Application

This Work Report Viewer application is designed for Gulf Petrochemical Services LLC to manage and visualize daily work reports submitted by field teams.

Features:

  • View daily work reports filtered by date and location
  • Track manpower and equipment utilization
  • Monitor work progress across different project areas
  • Access historical data for reporting and analysis

Instructions:

  1. Select a date using the date picker
  2. Filter by specific area/block if needed
  3. View detailed work reports with resource allocation
